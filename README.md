# FLIRHack
FLIR Hackathon

Instructions:  
  
1) Install modified FLIR SDK Demo app  
  
2) Start app  
  
3) Press Simulation mode button if you do not have a FLIR One  
  
4) Press camera button to save a picture  
  
5) Install Unity app  
  
6) Run Unity App  
  
7) Thermal image is displayed
